    \hypertarget{conclusion}{%
\subsection{Conclusion}\label{conclusion}}

\textbf{What are your key takeaways from this lab?}

    \begin{itemize}
\tightlist
\item
  There are many useful operators in Python that help you work with
  variables.

  \begin{itemize}
  \tightlist
  \item
    The \texttt{=} assignment operator allows you to assign or reassign
    a specific value to a variable.
  \end{itemize}
\item
  The \texttt{print()} function in Python allows you to display
  information.

  \begin{itemize}
  \tightlist
  \item
    It can take in a value directly or a variable that stores a value.
  \end{itemize}
\item
  The \texttt{type()} function in Python helps you to determine the data
  type of an object.

  \begin{itemize}
  \tightlist
  \item
    If you pass in a variable to \texttt{type()}, it will output the
    data type of the value stored in the variable.
  \end{itemize}
\item
  Python is able to convert some data types from one to another.

  \begin{itemize}
  \tightlist
  \item
    Using the \texttt{int()} function on a variable that stores data as
    a \textbf{float} will convert the data from a floating point number
    to an integer.
  \end{itemize}
\end{itemize}
